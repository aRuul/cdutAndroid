# cdutAndroid
cdut 安卓的课设📱

#### 连连看

实现的功能

- 连连看基本的消除
- 游戏困难等级（简单、普通、困难）
- 背景音乐
- 点击音效、消除音效
- 偏好设置
- sqlite数据库存储排行榜
- 游戏倒计时
- 隐藏彩蛋【彩蛋自己找吧】

---
